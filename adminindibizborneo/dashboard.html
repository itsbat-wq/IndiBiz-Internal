Format Template Excel

File Excel yang diunggah harus memiliki kolom-kolom berikut (urutan bebas):

Nama Kolom Tipe Data Contoh Keterangan
nama Text CV. Sinar Mandiri Nama lengkap pelanggan
no_internet Text/Number 1234567890 Nomor internet IndiBiz
kota Text Banjarmasin Kota pelanggan
paket Text IndiBiz High Speed 100Mbps Nama paket langganan
bulan Text April 2025 Bulan tagihan
tagihan Number 1250000 Nominal tagihan (angka saja)
status_bayar Text Lunas / Belum "Lunas" atau "Belum"
tanggal_bayar Number 5 Tanggal bayar (1-31). Poin: tgl 1-10 = x3, tgl 11-15 = x2, tgl 16-20 = x1
